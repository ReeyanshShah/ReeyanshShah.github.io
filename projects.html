3D DataCollector for IPEC

Wadhwani Electronics Lab, Guide - Siddhart Tallur

iPEC is a handheld device designed for instant measurement of rebar diameter and cover thickness in RCC structures. My role involved enhancing a 3D data collector setup, integrating motion control in X, Y, and Z axes using 3D printing and KiCad-designed PCBs. I developed an Arduino-based controller and PyQt GUI for automated scanning and precise data analysis, ensuring efficient structural inspections.

IITB-RISC Microprocessor Design

Course Project, EE 309, Prof. Virendra Singh

We developed a 6-stage pipeline processor to enhance instruction processing throughput and maximize cycle frequency, incorporating stages like Instruction Fetch, Instruction Decode, Register Read, Execute, Memory Access, and Writeback. We implemented advanced features such as a 2-level data forwarding logic and a history bit-based branch predictor to minimize data hazards and improve branch prediction accuracy, ensuring smooth operation without stalls. Additionally, we integrated hazard mitigation units to optimize performance and ensure smooth operation of the pipeline processor. We utilized VHDL modeling to implement the datapath design and control unit, and simulated the design using Quartus.

Wearable Exoskeleton Glove for Hand Motion Capture

As part of a team project, I helped develop a 3D-printed exoskeleton glove designed to capture and simulate real-time finger movements. The glove uses rotary encoders to measure joint angles and transmits data via Wi-Fi (UDP) using a Raspberry Pi Pico W and a custom PCB with multiplexers. This data is used to drive a 3D hand model in Unity, accurately replicating user finger motions with sub-second latency. The glove supports wireless operation, real-time tracking, and has potential applications in VR gaming, rehabilitation, and robotic control.

3D Data Collector for IPEC

Wadhwani Electronic Lab, Prof. Siddhart Tallur

iPEC is a handheld device designed for instant measurement of rebar diameter and cover thickness in RCC structures. My role involved enhancing a 3D data collector setup, integrating motion control in X, Y, and Z axes using 3D printing and KiCad-designed PCBs. I developed an Arduino-based controller and PyQt GUI for automated scanning and precise data analysis.

IITB-RISC Microprocessor Design

Course Project, EE 309, Prof. Virendra Singh

We developed a 6-stage pipeline processor to enhance instruction processing throughput and maximize cycle frequency, incorporating stages like Instruction Fetch, Instruction Decode, Register Read, Execute, Memory Access, and Writeback. We implemented advanced features such as a 2-level data forwarding logic and a history bit-based branch predictor to minimize data hazards and improve branch prediction accuracy, ensuring smooth operation without stalls. Additionally, we integrated hazard mitigation units to optimize performance and ensure smooth operation of the pipeline processor. We utilized VHDL modeling to implement the datapath design and control unit, and simulated the design using Quartus.

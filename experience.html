Embedded Software Engineer [May 2025 - Jul 2025]

Over two months at Texas Instruments, I worked with the Jacinto Baseport Team to port RHIVOS (Red Hat In-Vehicle Operating System) onto TI’s AM69-SK and J784S4 automotive evaluation boards. RHIVOS is a hardened automotive-grade OS based on RHEL and Automotive Grade Linux, tailored for safety-critical applications. My primary focus was understanding and adapting Red Hat’s build infrastructure to support TI hardware. I configured the Linux kernel, U-Boot bootloader, and Device Tree Source (DTS) to enable board-specific features. I also resolved boot-time networking issues by identifying and integrating essential RPM packages required for dnf and internet connectivity. I enabled and validated the display subsystem using Weston as the Wayland compositor, successfully running containerized graphical applications such as the Swell Foop game. To demonstrate virtualization capabilities, I deployed Alpine Linux inside a QEMU virtual machine. I also implemented a GStreamer-based camera-to-display pipeline that streamed real-time IP camera footage to the display. Additionally, I extended the open-source diagnostic tool k3conf with two custom modules: Basic Address Testing (BAT): Analyzed memory pages of running processes using pread() to validate page accessibility. Memory Access Test (MAT): Evaluated file-level access behavior by testing various permission scenarios programmatically.
